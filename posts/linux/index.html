    A Linux dead simple guide!

    Introduction What is a Linux distribution Linux distributions usually refers to a family of operating systems that uses the Linux kernel and are similar in some design aspects. For the most part, Linux distributions follows the Unix philosophy and are Unix-like operating systems. Distro is just a shorthand for distribution! Does it even matter So little, most things that can be done on Linux can be done on all distros. Distros are more of an entry point, how you would like your system to look like when you start using it.
